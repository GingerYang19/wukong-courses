📖 详细课程大纲

DAY1：商机经营与项目识局

上午：三网情报采集与早期机会识别

模块一：商机评分与入池排序

  • 1.1 线索真实性判断：来源可信度、项目动因分析
  • 1.2 项目价值评估：赢面基础、资源投入优先级
  • 1.3 实战演练：生成《大客户商机评分表》与《重点项目优先级矩阵》

模块二：决策链地图与关键人画像

  • 2.1 客户组织结构解析：使用层、技术层、采购层、管理层
  • 2.2 关键角色识别：推动者、影响者、审慎者、否决者与Coach教练
  • 2.3 实战演练：生成《决策链地图》与《关键人画像表》

下午：招标规则拆解与竞对打法判断

模块三：招标规则拆解与得分策略设计

  • 3.1 招标文件深度解析：高权重得分点、一票否决项
  • 3.2 技商一致性风险检查：技术方案与商务条款匹配度
  • 3.3 实战演练：生成《评分拆解表》与《得分策略设计》

模块四：竞对打法识别与赢面变化判断

  • 4.1 竞争对手画像：核心优势、过往中标记录、价格策略
  • 4.2 赢面动态评估：竞争态势分析与应对动作设计
  • 4.3 实战演练：生成《竞对打法分析表》与《应对动作表》

DAY2：需求推进与客户深耕

上午：客户压力诊断与加速变量识别

模块五：客户压力诊断与需求访谈纪要

  • 5.1 客户痛点深挖：业务压力、预算信号、时间窗口
  • 5.2 立项动力诊断：从“有兴趣”到“真立项”的转化路径
  • 5.3 实战演练：生成《客户压力诊断表》与《需求访谈纪要》

模块六：加速变量识别与推进节奏计划

  • 6.1 项目加速变量：内部推动者、外部政策、竞争压力

  • 6.3 实战演练：生成《加速变量表》与《30天推进节奏计划》

🎯 Demo场景

Demo 1：三网情报采集与早期机会识别

从政策、招采、客户动态中发现早期商机。利用悟空解析《三网情报资料包》，按情报网、关系网、渠道网分类，筛出最值得验证的早期机会。

Demo 2：商机评分与入池排序

对多条线索进行真实性判断、项目价值评估和赢面基础分析。生成《大客户商机评分表》与《重点项目优先级矩阵》，决定资源投入优先级。

Demo 3：决策链地图与关键人画像

基于客户沟通记录，识别使用部门、技术部门、采购部门、管理层等关键角色，区分推动者、影响者、审慎者、否决者，生成《决策链地图》。

Demo 4：招标规则拆解与得分策略设计

深度解析招标文件，识别高权重得分点、一票否决项及技商一致性风险，生成《评分拆解表》与《得分策略设计》。

Demo 5：竞对打法识别与赢面变化判断

分析竞争对手的核心优势、过往中标记录及价格策略，动态评估赢面变化，生成《竞对打法分析表》与《应对动作表》。

Demo 6：客户压力诊断与需求访谈纪要

深挖客户业务压力、预算信号与时间窗口，诊断立项动力，将拜访纪要转化为《客户压力诊断表》与《需求访谈纪要》。
